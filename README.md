# Exam Organisation System

The application aims to help students and teachers easily access their exam information, classroom details, seating order and announcements from a single platform.

# Features

- Student Login
- Home Dashboard
- Exam List
- Exam Detail Page
- QR Demonstration
- Seat Information Page
- Student Profile
- Settings Page
- Announcements (planned)
- Dark Mode (planned)
- Teacher Login (planned)

---

# Technologies

- .NET MAUI (.NET 10)
- C#
- XAML
- JetBrains Rider
- MVVM (In Progress)
- SQLite (Planned)
